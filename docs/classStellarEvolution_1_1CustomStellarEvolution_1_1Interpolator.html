Creates an interpolator from a given track.

Parameters
filenameThe filename of the track.
formatA list of the columns in the track
smoothingThe smoothing to apply to each column (NaN for no smoothing).

Entries for AGE and SKIP are ignored.

Parameters
nodesThe number of nodes to use when smoothing each column.

This is ignored for columns for which the corresponding entry in smoothing is NaN

Negative entries result in using the smaller of the absolute value of the entry and three times the number of track points.

Parameters
vs_log_ageShould interpolation be done vs. log(age) instead of age for each quantity?
log_quantityShould interpolation be done of log(quantity) instead of quantity for each quantity?
